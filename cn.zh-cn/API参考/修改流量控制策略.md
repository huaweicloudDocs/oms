# 修改流量控制策略<a name="ZH-CN_TOPIC_0188149529"></a>

## 示例<a name="section1771037105620"></a>

**请求示例**

PUT /v1/\{project\_id\}/objectstorage/task/\{task\_id\}

```
{
  "operation": "change_bandwidthpolicy",
  "bandwidth_policy": [
    {
      "start": "00:00",
      "end": "23:59",
      "max_bandwidth": 50000000
    }
  ]
}
```

**响应示例**

无。
