## 示例<a name="section17067371"></a>

**请求示例**

GET /v1/\{domain\_id\}/objectstorage/quotas

**响应示例**

```
{
  "quotas": {
    "resource": {
      "type": "MaxParallelTaskNum",
      "quota": 5
    }
  }
}
```
