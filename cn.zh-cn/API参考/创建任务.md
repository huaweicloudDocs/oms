# 创建任务<a name="oms_api_0014"></a>

## 功能介绍<a name="section1598569"></a>

调用该接口添加迁移任务。创建任务成功后，任务会被自动启动，不需要额外调用启动任务命令。

注意事项：

-   单个租户在过去24小时内最多可创建200个迁移任务。
-   单个租户名下处于“等待中”状态的迁移任务达到50个时不可再创建任务。

## URI<a name="section14387121"></a>
